# Android-Project--Recipe-Book

An Android application that lets users browse, view, and manage recipes in a clean and user-friendly interface. The app is designed as a simple recipe book where users can explore different dishes, view ingredients and cooking steps, and enjoy an organized cooking experience on mobile.
